# Задача 2 - Работа с массивами и OpenMP

## Описание
В данной работе реализована программа на C++, которая находит минимальное и
максимальное значения массива двумя способами:
- последовательным
- параллельным с использованием OpenMP

Также выполняется сравнение времени работы обеих реализаций.

## Условия задания
- Создать массив из 10 000 случайных чисел  
- Найти минимальное и максимальное значения:
  - последовательным способом
  - с использованием OpenMP  
- Сравнить время выполнения  
- Сделать выводы  

## Логика работы программы
1. Создаётся массив из 10 000 элементов  
2. Массив заполняется случайными числами  
3. Последовательно находится min и max  
4. Измеряется время выполнения  
5. С использованием OpenMP выполняется параллельный поиск  
6. Измеряется время выполнения  
7. Результаты выводятся в консоль  
8. Память освобождается  

## Файлы

- `block.png` - блок-схема алгоритма (текст)
