# Задача 1 — Введение в гетерогенную параллелизацию

## Описание
Гетерогенная параллелизация — это подход к программированию, при котором
вычисления распределяются между разными типами вычислительных устройств,
например CPU и GPU, с целью повышения производительности.

## Параллельные вычисления на CPU и GPU

### CPU
- Имеет небольшое количество мощных ядер  
- Хорошо подходит для сложной логики и ветвлений  
- Используется для управления программой и последовательных задач  

### GPU
- Содержит тысячи простых вычислительных ядер  
- Эффективен для выполнения одинаковых операций над большими массивами данных  
- Лучше всего подходит для массовых параллельных вычислений  

## Преимущества гетерогенной параллелизации
- Повышение производительности за счёт использования GPU  
- Более эффективное использование ресурсов компьютера  
- Возможность распределять задачи между CPU и GPU в зависимости от их типа  
- Ускорение обработки больших объёмов данных  

## Примеры применения
Гетерогенная параллелизация широко используется в следующих областях:
- обучение нейронных сетей и машинное обучение  
- обработка изображений и видео  
- научные и инженерные расчёты  
- компьютерная графика и визуализация  
- криптография и блокчейн-вычисления  

## Вывод
Гетерогенная параллелизация позволяет значительно ускорить выполнение
вычислительно сложных задач за счёт совместного использования CPU и GPU.
Эффективность данного подхода особенно заметна при работе с большими объёмами данных.
