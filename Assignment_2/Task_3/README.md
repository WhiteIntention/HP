# Задача 3 - Параллельная сортировка выбором (OpenMP)

## Описание
В работе реализована сортировка выбором (Selection Sort) двумя способами:
- последовательная версия
- версия с OpenMP (параллельный поиск минимального элемента)

Производительность проверяется для массивов размером 1000 и 10000 элементов.

## Условия задания
- Реализовать последовательную сортировку выбором  
- Добавить параллелизм с использованием OpenMP  
- Проверить время работы для N = 1000 и N = 10000  

## Как работает параллельная версия
В сортировке выбором внешний цикл по i остаётся последовательным.
Параллелизм добавляется внутри каждой итерации: поиск минимального элемента
на отрезке [i..n-1] выполняется несколькими потоками.
