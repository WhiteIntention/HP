# Задача 4 - Сортировка на GPU (CUDA)

## Описание
Реализована параллельная сортировка слиянием на GPU с использованием CUDA.
Массив делится на подмассивы, каждый подмассив обрабатывается отдельным блоком,
после чего выполняются параллельные проходы слияния.

## Условия задания
- Разделить массив на подмассивы, каждый сортируется отдельным блоком
- Выполнить параллельное слияние отсортированных подмассивов
- Замерить производительность для N = 10 000 и N = 100 000

## Как работает программа
1. На CPU создаётся массив и заполняется случайными числами
2. Массив копируется на GPU
3. Каждый блок сортирует свой кусок (chunk)
4. Дальше выполняются проходы слияния: run удваивается (chunk, 2*chunk, 4*chunk, ...)
5. Результат копируется обратно на CPU
6. Проверяется, что массив отсортирован
7. Выводится время выполнения

## Файлы
