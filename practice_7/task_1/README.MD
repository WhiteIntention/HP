# Задание 7.1 - Реализация редукции на CUDA

## Описание
В данной работе реализована операция **редукции (суммирования элементов массива)** с использованием технологии CUDA.  
Суммирование выполняется на GPU с применением **разделяемой памяти (shared memory)** для оптимизации доступа к данным и снижения количества обращений к глобальной памяти.

Для проверки корректности полученный результат сравнивается с эталонной реализацией на CPU.

## Постановка задачи
1. Реализовать CUDA-ядро для редукции (суммирования элементов массива).  
2. Использовать shared memory для оптимизации вычислений.  
3. Проверить корректность работы на тестовых массивах.

## Реализация
- Каждому блоку CUDA соответствует вычисление частичной суммы.
- Каждый поток обрабатывает до двух элементов входного массива.
- Частичные суммы сохраняются в shared memory.
- Внутри блока выполняется поэтапная бинарная редукция.
- Итоговая сумма вычисляется в несколько проходов (многошаговая редукция).
- Для проверки корректности используется CPU-реализация суммирования.

## Тестовые данные
Для тестирования использовались массивы случайных чисел следующих размеров:
- `1024` элементов  
- `1 000 000` элементов  
- `10 000 000` элементов  

## Результаты

### N = 1024
- **CPU sum:** 506.488054  
- **GPU sum:** 506.488037  
- **Абсолютная ошибка:** 0.000017  
- **Относительная ошибка:** ≈ 0  
- **Время GPU:** 0.100320 ms  

### N = 1 000 000
- **CPU sum:** 500585.411479  
- **GPU sum:** 500585.375000  
- **Абсолютная ошибка:** 0.036479  
- **Относительная ошибка:** ≈ 0  
- **Время GPU:** 0.075168 ms  

### N = 10 000 000
- **CPU sum:** 5000375.425453  
- **GPU sum:** 5000376.000000  
- **Абсолютная ошибка:** 0.574547  
- **Относительная ошибка:** ≈ 0  
- **Время GPU:** 0.539136 ms  

## Анализ результатов
- Абсолютная погрешность увеличивается с ростом размера массива, что связано с:
  - использованием типа `float` при вычислениях на GPU;
  - различным порядком суммирования элементов на CPU и GPU.
- Относительная ошибка во всех тестах близка к нулю, что подтверждает корректность реализации.
- GPU демонстрирует высокую производительность при обработке больших массивов данных.
