# Часть 2 - Структуры данных + OpenMP

## Описание
Реализованы структуры данных:
- односвязный список (добавление, удаление, поиск)
- стек (push, pop, isEmpty)
- очередь (добавление в конец, удаление из начала, проверка на пустоту)

Также добавлено параллельное выполнение некоторых операций с OpenMP:
- параллельное добавление элементов в список и очередь
- сравнение времени последовательного и параллельного добавления

## Файлы
- `main.cpp` — код программы
- `block.png` — блок-схема (текст)
- `screenshot.png` — вывод в консоли
