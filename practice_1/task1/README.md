# Часть 1 - Работа с массивами (OpenMP)

## Описание
Программа на C++:
1) создаёт массив из N элементов и заполняет случайными числами от 1 до 100  
2) находит минимальный и максимальный элементы
- последовательно
- параллельно с OpenMP  
3) выводит массив, min/max и время выполнения обеих реализаций

## Файлы
- `main.cpp` - код программы
- `block.png` - блок-схема
- `screenshot.png` - скрин вывода в консоли

## Сборка и запуск

### GCC / MinGW (OpenMP)
```bash
g++ -O2 -std=c++17 -fopenmp main.cpp -o part1.exe
./part1.exe
