# Часть 3 - Динамическая память и указатели (OpenMP reduction)

## Описание
Программа создаёт динамический массив через указатель, заполняет его случайными
числами и вычисляет среднее значение:
- последовательным способом
- параллельно с использованием OpenMP и reduction(+:sum)

Также сравнивается время выполнения.

## Условия
- Создать динамический массив и заполнить случайными числами
- Написать функцию для поиска среднего значения
- Использовать OpenMP: `#pragma omp parallel for reduction(+:sum)`
- Освободить память после работы

## Файлы
- `main.cpp` — код программы
- `block.png` — блок-схема
- `screenshot.png` — вывод в консоли
