# Практическая работа 2 - Сортировки с OpenMP

## Описание
Реализованы сортировки:
- пузырьком
- выбором
- вставками

Сделаны последовательные версии и версии с OpenMP.
Проверка производительности проводится на массивах: 1000, 10000, 100000 элементов.

## Важно
Классические bubble/selection/insertion имеют зависимости между шагами, поэтому
их прямое распараллеливание по внешнему циклу обычно не даёт хорошего ускорения.

В программе:
- пузырёк: использован корректный параллельный вариант odd-even (параллельные сравнения)
- выбор: внешний цикл последовательный, а поиск минимума внутри распараллелен
- вставки: сортируются куски массива параллельно, потом выполняется слияние
