# Практическая работа 2 - Реализация сортировок без параллелизма

## Описание
В данной работе реализованы три классических алгоритма сортировки
без использования параллельных технологий:
- пузырьковая сортировка
- сортировка выбором
- сортировка вставками

Все алгоритмы реализованы на языке C++ и работают последовательно.

## Условия задания
- Реализовать функции сортировки пузырьком, выбором и вставками
- Не использовать OpenMP и другие средства параллелизации
- Продемонстрировать работу алгоритмов на одном массиве

## Логика работы программы
1. Создаётся массив фиксированного размера
2. Массив заполняется случайными числами
3. Выводится исходный массив
4. Массив копируется для каждой сортировки
5. Выполняются три вида сортировки
6. Выводятся отсортированные массивы
7. Освобождается динамическая память
