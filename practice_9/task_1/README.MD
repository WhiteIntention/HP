# Задание 9.1 — Распределённое вычисление среднего значения и стандартного отклонения (MPI)

## Цель работы
Изучить использование продвинутых операций MPI для распределённой обработки
данных, а именно:
- передачу данных с учётом неравномерного распределения (`MPI_Scatterv`);
- коллективные операции для агрегации результатов (`MPI_Reduce`);
- корректную работу программы при любом количестве процессов.

---

## Описание реализации
Программа выполняет следующие шаги:
1. На процессе с `rank = 0` создаётся массив случайных чисел размера `N`.
2. Массив распределяется между всеми процессами с помощью `MPI_Scatterv`,
   что позволяет корректно обработать случай, когда `N` не делится нацело
   на количество процессов.
3. Каждый процесс вычисляет для своей части массива:
   - сумму элементов;
   - сумму квадратов элементов.
4. Локальные суммы собираются на процессе `rank = 0` с использованием
   `MPI_Reduce`.
5. На основе собранных данных вычисляются:
   - среднее значение массива;
   - стандартное отклонение.
6. Результаты выводятся на экран процессом `rank = 0`.

---

## Тестовые данные
Эксперименты были проведены для двух размеров массива:
- `N = 1 000 000`
- `N = 10 000 000`

Количество MPI-процессов: `4`.

---

## Результаты

### N = 1 000 000
- MPI processes = `4`
- Mean = `0.500094`
- StdDev = `0.288560`

### N = 10 000 000
- MPI processes = `4`
- Mean = `0.500053`
- StdDev = `0.288706`

---

## Анализ результатов
- Полученные значения среднего близки к `0.5`, что соответствует ожидаемому
  значению для равномерного распределения на интервале `[0, 1)`.
- Значения стандартного отклонения близки к теоретическому значению
  `√(1/12) ≈ 0.288675`, что подтверждает корректность вычислений.
- Совпадение результатов для разных размеров массива показывает, что
  программа корректно масштабируется и не зависит от разбиения данных.
- Использование `MPI_Scatterv` обеспечивает корректную работу программы
  при любом количестве процессов, даже если размер массива не делится
  нацело.

---

## Вывод
В ходе задания была реализована распределённая программа с использованием
MPI, которая корректно вычисляет среднее значение и стандартное отклонение
массива данных. Эксперименты подтвердили правильность работы коллективных
операций `MPI_Scatterv` и `MPI_Reduce`, а также устойчивость алгоритма при
увеличении размера входных данных.
