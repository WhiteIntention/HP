# Задание 8.2 — Обработка массива на GPU с использованием CUDA

## Цель
Реализовать обработку массива данных на GPU с помощью CUDA и замерить время выполнения.
Операция обработки: **умножить каждый элемент массива на 2**.

---

## Что сделано
1. На CPU создаётся массив случайных чисел размера `N`.
2. Массив копируется на GPU (Host → Device).
3. Запускается CUDA-ядро, которое умножает каждый элемент на 2.
4. Результат копируется обратно на CPU (Device → Host).
5. Замеряется время:
   - копирования **H2D**
   - выполнения **kernel**
   - копирования **D2H**
   - итоговое время (сумма трёх этапов)

---

## Тестовые данные
Проверка выполнена для двух размеров массива:
- `N = 1 000 000`
- `N = 10 000 000`

---

## Результаты

### N = 1 000 000
- First element = `1.392938`  
- GPU H2D time (ms) = `1.060000`  
- GPU kernel time (ms) = `0.188704`  
- GPU D2H time (ms) = `0.972576`  
- GPU total time (ms) = `2.221280`  

### N = 10 000 000
- First element = `0.784892`  
- GPU H2D time (ms) = `9.853184`  
- GPU kernel time (ms) = `0.360224`  
- GPU D2H time (ms) = `8.733856`  
- GPU total time (ms) = `18.947264`  

---

## Короткий анализ
- Время копирования данных **H2D + D2H** занимает основную часть общего времени.
- Время выполнения ядра (kernel) относительно маленькое даже для `10 000 000` элементов.
- При увеличении размера массива время kernel растёт не так сильно, как время передачи данных, поэтому узкое место здесь — именно копирование CPU ↔ GPU.
- Вывод первого элемента после обработки показывает, что операция умножения на 2 выполнена корректно.
