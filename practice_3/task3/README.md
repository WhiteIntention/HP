# CUDA - Параллельная пирамидальная сортировка (Heap Sort)

## Задание
Реализовать пирамидальную сортировку на CUDA:
- построить кучу;
- выполнять извлечение элементов параллельно, где это возможно.

## Идея решения
Heap Sort состоит из двух этапов:

1) **Построение max-heap (heapify)**  
   Этот этап можно распараллелить частично: узлы на одном уровне дерева
   просеиваются независимо. Поэтому heapify выполняется на GPU по уровням
   снизу вверх с помощью CUDA-ядер.

2) **Извлечение элементов**  
   Этот этап зависит от предыдущих шагов (после каждого извлечения меняется куча),
   поэтому полностью параллельным его сделать сложно.  
   Чтобы избежать огромного количества запусков kernel, извлечение и просеивание
   выполнено внутри **одного CUDA-ядра** (в одном потоке), но всё равно на GPU.

## Что используется
- C++
- CUDA kernels
- замер времени через `cudaEvent`
