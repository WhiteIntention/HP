# CUDA - Параллельная быстрая сортировка (QuickSort)

## Описание задания
В данной работе реализована параллельная быстрая сортировка на GPU с
использованием технологии **CUDA**.

Требования задания:
- использовать параллельные потоки для разбиения массива по опорному элементу;
- каждая часть массива сортируется отдельно;
- реализовать алгоритм без использования OpenMP (только CUDA).

---

## Описание решения
Реализация основана на алгоритме **QuickSort** с параллельным разбиением
(partition) массива на GPU.

Основные этапы алгоритма:
1. Массив копируется из оперативной памяти (CPU) в память GPU.
2. Для текущего сегмента выбирается опорный элемент (pivot).
3. Параллельный CUDA-ядро формирует массив флагов, определяющих элементы,
   меньшие опорного.
4. С помощью параллельного префиксного суммирования вычисляются позиции элементов.
5. Элементы перераспределяются (scatter) в соответствии с результатами разбиения.
6. Полученные подмассивы обрабатываются аналогичным образом.
7. Для маленьких сегментов используется быстрая сортировка на GPU.
8. Отсортированный массив копируется обратно на CPU.

Алгоритм реализован итеративно (без рекурсии на GPU) с использованием стека сегментов.

---

## Параллелизм
- Разбиение массива по pivot выполняется **параллельно** с использованием CUDA-ядер.
- Каждый поток обрабатывает свою часть данных.
- Параллельное префиксное суммирование используется для корректного размещения элементов.
- Малые сегменты сортируются отдельно, что снижает накладные расходы.
