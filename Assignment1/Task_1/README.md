Assignment 1.1  
Динамический массив и вычисление среднего значения

Описание
В данном задании реализована работа с динамическим массивом в языке C++.
Программа создаёт массив фиксированного размера, заполняет его случайными
числами и вычисляет среднее значение элементов.

Условия задания
- Создать динамический массив из 50 000 целых чисел  
- Заполнить массив случайными значениями в диапазоне от 1 до 100  
- Вычислить среднее значение элементов массива  
- Освободить выделенную память  

Логика работы программы
1. Выделяется память под динамический массив  
2. Инициализируется генератор случайных чисел  
3. Массив заполняется случайными значениями  
4. Подсчитывается сумма элементов массива  
5. Вычисляется среднее значение  
6. Результаты выводятся в консоль  
7. Память освобождается  

Файлы в папке
- `main.cpp` — исходный код программы  
- `screenshot.png` — результат выполнения программы (консоль)  
- `block.png` — блок-схема алгоритма  

Компиляция и запуск
GCC / MinGW
g++ main.cpp -fopenmp -O2 -std=c++17 -o main.exe
.\main.exe  

Вывод
Программа корректно вычисляет среднее значение элементов динамического массива.
Результаты подтверждают правильность работы алгоритма и управления памятью.
