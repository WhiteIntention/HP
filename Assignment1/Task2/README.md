# Assignment 1.2  
Последовательный поиск минимального и максимального значений

## Описание
В данном задании реализуется последовательная обработка массива для поиска
минимального и максимального элементов. Также измеряется время выполнения
алгоритма.

## Условия задания
- Создать динамический массив из 1 000 000 целых чисел  
- Заполнить массив случайными значениями  
- Найти минимальное и максимальное значения последовательным способом  
- Измерить время выполнения алгоритма  
- Освободить выделенную память  

## Логика работы программы
1. Выделяется память под динамический массив  
2. Массив заполняется случайными числами  
3. Запускается измерение времени  
4. Последовательно выполняется поиск минимума и максимума  
5. Останавливается измерение времени  
6. Результаты выводятся в консоль  
7. Память освобождается  

## Файлы в папке
- `main.cpp` — исходный код программы  
- `screenshot.png` — результат выполнения программы (консоль)  
- `block.png` — блок-схема алгоритма  

## Компиляция и запуск
### GCC / MinGW
```bash
g++ main.cpp -fopenmp -O2 -std=c++17 -o main.exe
.\main.exe
```
## Вывод
Программа корректно находит минимальное и максимальное значения массива.
Время выполнения показывает высокую эффективность последовательного алгоритма
на современных процессорах.
