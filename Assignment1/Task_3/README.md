# Assignment 1.3  
Параллельный поиск минимального и максимального значений (OpenMP)

## Описание
В данном задании реализуется параллельная обработка массива с использованием
технологии OpenMP. Программа находит минимальное и максимальное значения
элементов массива и измеряет время выполнения алгоритма.

## Условия задания
- Создать динамический массив из 1 000 000 целых чисел  
- Заполнить массив случайными значениями  
- Найти минимальное и максимальное значения с использованием OpenMP  
- Измерить время выполнения  
- Освободить выделенную память  

## Логика работы программы
1. Выделяется память под динамический массив  
2. Массив заполняется случайными числами  
3. Запускается измерение времени  
4. Создаётся параллельная область OpenMP  
5. Каждый поток вычисляет локальные минимум и максимум  
6. Локальные результаты объединяются в глобальные значения  
7. Останавливается измерение времени  
8. Результаты выводятся в консоль  
9. Память освобождается  

## Файлы в папке
- `main.cpp` — исходный код программы (OpenMP)  
- `screenshot.png` — результат выполнения программы (консоль)  
- `block.png` — блок-схема алгоритма  

## Компиляция и запуск
### GCC / MinGW (OpenMP)
```bash
g++ main.cpp -fopenmp -O2 -std=c++17 -o main.exe
.\main.exe
```

## Вывод
Программа корректно выполняет параллельный поиск минимального и максимального
значений массива. Полученные результаты совпадают с последовательной версией,
что подтверждает корректность параллельного алгоритма.
